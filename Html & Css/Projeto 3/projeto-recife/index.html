Marco Zero

O Marco Zero é um dos pontos turísticos mais emblemáticos de Recife. Localizado no bairro do Recife Antigo, ele marca o início das medições das distâncias das estradas de Pernambuco. É um local cheio de história, cultura e beleza, com vista para o Rio Capibaribe.

Instituto Ricardo Brennand

O Instituto Ricardo Brennand é um espaço cultural que abriga um acervo permanente de obras de arte, armaduras, esculturas e muito mais. Localizado em um belíssimo castelo, o instituto é um passeio imperdível para quem visita Recife, oferecendo uma experiência única de arte e história.
